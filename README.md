## 1일차 학습 내용
- 웹 기본 개념

    - 웹 동작 => "request & response"
- 웹 개발 표준 기술(HTML,CSS,Javascript)

        - HTML5 기본 태그
            - head 태그 => meta, title, link, style, script 등
            - a 태그 => 절대 경로, 상대 경로 등
            - list 태그 => ul, ol, li
            - media 태그 => img, audio, video
